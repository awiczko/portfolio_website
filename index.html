What I have learnt

Administration and management of Microsoft SQL servers

This course provided essential skills for managing and maintaining SQL servers. I learned how to install and configure SQL servers for optimal performance, implement backup strategies, and restore databases. I gained experience in importing/exporting data, monitoring server activity, and ensuring high availability. The course also covered SQL server security, ongoing database maintenance, and automation to improve efficiency and minimize manual work.

Laser scanning TLS, MLS, and UAV technologies

This training covers the theoretical foundations of laser scanning and photogrammetry technologies, as well as an overview of drone types and their applications. It includes practical sessions on both stationary and mobile scanning, working with point clouds (registration, editing, data integration, and result publication), and practical photogrammetry exercises such as creating point clouds and textured mesh models from images.

Linux Command Line Basics

This course introduces the basics of using the Linux command line. It covers essential commands for navigating the file system, managing files and directories, and performing basic system operations. The training also includes tips for using text editors, manipulating files, and executing scripts, providing a solid foundation for efficient command line usage.
